Pertanyaan yang sering ditanya

Benar-benar gratis? Nggak ada hidden cost? +
Ya, 100% gratis. Saat ini fynni dalam fase testing dengan user terbatas. Semua fitur inti bisa kamu pakai tanpa bayar sepeser pun.
Aman nggak data keuangan saya? +
Sangat aman. Data dienkripsi end-to-end dan hanya tersimpan di device kamu. Kami tidak menjual data keuangan ke pihak ketiga.
Ribet nggak setup awalnya? +
Nggak. Setup kurang dari 2 menit. Cukup download, buat akun, dan mulai input transaksi pertama.
Bisa sinkron dengan bank atau e-wallet? +
Saat ini fynni fokus pada input manual untuk kecepatan dan keamanan maksimal. Integrasi otomatis ada di roadmap.
Apa bedanya fynni dengan budget tracker lain? +
fynni punya speed-to-insight ≤3 detik, UI dark-first, dan rekomendasi budget berbasis panduan profesional.
Bisa pakai di web atau hanya mobile? +
Saat ini dioptimalkan untuk mobile. Versi web sedang dalam pengembangan.
